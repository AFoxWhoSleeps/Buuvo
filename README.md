# BUUVO

**COMMANDS** / case-sensitive:
> ```%``` (Used to execute a command of your choice / see below)

> ```switch``` (Used to switch between functions A, B, C, and/or D)

> ```%switch Function A```

> ```%switch Function B```

> ```%switch Function C```

> ```%switch Function D```

> ```exit``` (```%``` is not used)
